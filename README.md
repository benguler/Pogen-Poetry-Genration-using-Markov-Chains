# Pogen-Poetry-Genration-using-Markov-Chains

PoGen is a simple poetry generator using Markov Chains. This goal of this project was to help writers find inspiration on poems of a specific genre. Users have the option to choose the genre as well as the syllable scheme of the poem.

## Dependencies
- nltk
```
pip install nltk
```
- textblob
```
pip install -U textblob
python -m textblob.download_corpora
```
- syllables
```
pip install syllables
```

## How to use the program?

Assuming all depencies are installed correctly run the program and follow the prompts to generate a poem

```
python PoGen.py
```

# There are a total of 6 poems genres

- Modern Love
- Modern Nature
- Modern Mythology & Folklore
- Renaissance Love
- Renaissance Nature
- Renaissance Mythology & Folklore

## Some poems examples generated by PoGen
